隱私權政策

新竹冬日童樂會（以下簡稱「本遊戲」或「應用程式」）是由 august8 明日互動 開發的增強實境（AR）圖卡角色收集遊戲。本隱私權政策說明我們如何處理使用者在遊戲中使用應用程式時的個人資料。本政策適用於本遊戲的 iOS 和 Android 版本，以及相關的 GitHub 儲存庫頁面。

我們致力於保護您的隱私權。本遊戲的設計原則是最小化資料收集：除了 AR 功能的必要權限（例如相機存取，用於掃描圖卡）和本地遊戲進度儲存以外，我們不會收集、儲存或傳輸任何個人可識別資訊（如姓名、電子郵件、位置、裝置 ID 等）。所有遊戲資料均儲存在您的裝置上，不會上傳至伺服器。

本隱私權政策自 2025 年 11 月 1 日 起生效。如有變更，我們將在本遊戲服務頁面公告，並更新生效日期。

1. 我們收集的資訊

本遊戲僅收集以下非個人資料，以確保 AR 功能和遊戲體驗順暢：

  • AR 功能必要權限：
    • 相機存取：用於掃描圖卡並疊加 AR 角色。影像僅在即時處理後即時刪除，不會儲存或上傳。
    • 陀螺儀/加速計：用於 AR 定位和互動，僅在遊戲運行時使用。
  • 遊戲進度資料：
    • 本地儲存的遊戲狀態，例如已收集的角色清單、等級進度、解鎖項目。
    • 這些資料儲存在您的裝置上（例如 iOS 的 Keychain 或 Android 的 SharedPreferences），不會傳輸至任何外部伺服器。

我們不收集：

  • 個人識別資訊（PII），如姓名、電子郵件、電話號碼。
  • 位置資料（除非 AR 掃描時的即時環境，無 GPS 追蹤）。
  • 聯絡人、照片庫或其他裝置資料。
  • 任何網路活動記錄或廣告追蹤 ID。

若您選擇透過 GitHub 回報問題或貢獻程式碼，我們可能間接接觸到您提供的公開資訊，但這不屬於本遊戲的資料收集範圍。

2. 我們如何使用您的資訊

收集的資料僅用於：

  • 提供 AR 圖卡掃描和角色互動功能。
  • 儲存和管理您的遊戲進度，讓您在裝置間（若支援雲端備份，由您的作業系統處理）恢復遊戲。
  • 改善遊戲體驗，例如偵錯或更新 AR 演算法（不涉及個人資料）。

我們不會使用這些資料進行：

  • 廣告個人化。
  • 與第三方分享。
  • 分析使用者行為模式。

3. 資訊分享與揭露

  • 不分享：我們不會將任何資料分享給第三方，包括廣告商、合作夥伴或分析服務。
  • 法律要求：僅在法律強制（如法庭命令）下，我們可能需揭露資料，但由於無個人資料，此情況極少發生。
  • GitHub 整合：若遊戲程式碼在 GitHub 上公開，任何程式碼貢獻均為公開的，不含使用者資料。

4. 資料安全

  • 所有本地資料使用裝置內建加密（例如 iOS 的資料保護 API 和 Android 的加密儲存）。
  • AR 影像處理在裝置端完成，無需網路傳輸。
  • 我們定期審核程式碼以確保無意外資料洩漏。

5. 兒童隱私

本遊戲名為「新竹冬日童樂會」，適合兒童使用。我們符合《兒童線上隱私保護法》（COPPA）及台灣《個人資料保護法》相關規定：

  • 不針對 13 歲以下兒童收集個人資料。
  • 家長可透過裝置設定管理權限（如相機存取）。
  • 若您懷疑兒童使用本遊戲，請聯絡我們以取得指導。

6. 您的權利

  • 存取與刪除：遊戲進度儲存在您的裝置，您可隨時清除應用程式以刪除所有資料。
  • 選擇退出：您可拒絕授予 AR 權限，但這可能影響遊戲功能。
  • 跨境傳輸：無資料傳輸，因此無跨境問題。

7. 政策變更

我們可能不時更新本政策，以反映遊戲變更或法規要求。變更將於服務頁面或遊戲內公告，至少提前 30 天通知。持續使用即表示同意變更。

8. 聯絡我們

如有隱私權問題，請透過電子郵件 [您的聯絡電子郵件，例如：privacy@yourdomain.com] 聯絡我們。我們將盡快回應（通常在 7 個工作日內）。
